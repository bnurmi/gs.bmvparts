# Replit Build Brief: Paid Subscriber AI Photo Quote Tool

Build a paid-subscriber-only page inside the existing BMW Parts Catalogue / Pricing system.

## Feature Name

AI Photo Quote Generator

## Purpose

Allow vendors or paid subscribers to upload accident/damage photos of a BMW, enter or detect the VIN, run AI damage analysis, match damaged/salvageable parts against the existing BMW parts catalogue, and generate a CSV quote in the same format as the existing G80 M3 pricing comparison output.

This should replicate the workflow already completed manually for the attached G80 M3 example.

## Access Control

This feature must only be available to paid subscribers.

Requirements:

- Check the user subscription status before allowing access
- Block unpaid users with an upgrade/paywall message
- Vendors should have access if their account has an active vendor/paid plan
- Admin users should always have access

## Page Requirements

Create a new page:

`/photo-quote`

or inside the vendor dashboard:

`/vendor/photo-quote`

The page should include:

1. Upload vehicle photos
2. Enter VIN manually
3. Optional: auto-detect VIN from uploaded images if visible
4. Select vehicle model if VIN is missing
5. Run AI damage analysis
6. Review detected damaged parts
7. Generate quote
8. Export CSV
9. Save quote to vendor/customer record

## AI Photo Analysis

The AI should inspect uploaded photos and detect:

- Vehicle make/model
- VIN if visible
- Front damage
- Rear damage
- Side damage
- Interior damage
- Airbag deployment
- Headlight/taillight damage
- Bumper damage
- Bonnet/boot damage
- Guard/fender damage
- Doors
- Mirrors
- Wheels
- Suspension visible damage
- Cooling pack/radiator/condenser damage
- Reinforcement/crash bar damage
- Trim pieces
- Any obvious missing parts

The AI should output a structured list of likely required parts.

Each detected part should include:

- OEM description
- Damage confidence score
- Damage location
- Suggested category
- Whether part is required, optional, or needs human review
- Notes from image analysis

## Catalogue Matching

Use the existing BMW parts catalogue and pricing engine to match detected parts to:

- BMW OEM description
- BMW OEM part number
- BMW new price in AUD
- M Performance Parts used price
- Saving amount
- Category

If VIN is available, use VIN-specific compatibility first.

If VIN is not available, fall back to model/year/platform matching.

## Vendor VIN API Quote Pull

Add an API endpoint so vendors can generate quotes directly via VIN.

Example endpoint:

`POST /api/vendor/photo-quote`

Input:

```json
{
  "vin": "WBS42AY000FT45458",
  "vehicle": "BMW G80 M3 Competition",
  "photos": ["uploaded_photo_urls"],
  "vendor_id": "vendor_123"
}
````

Output:

```json
{
  "quote_id": "quote_123",
  "vehicle": "BMW G80 M3 Competition",
  "vin": "WBS42AY000FT45458",
  "csv_url": "...",
  "detected_parts": [],
  "total_bmw_new": 0,
  "total_our_price": 0,
  "total_saving": 0
}
```

## CSV Output Format

The generated CSV must match the existing format shown in the attached screenshot.

Header section:

* BMW G80 M3 Competition Parts Pricing Comparison
* Vehicle
* VIN
* Date prepared
* M PERFORMANCE PARTS

Table columns:

* OEM Description
* BMW New (AUD)
* Our Price (AUD)
* Saving
* Category
* BMW OEM #

Pricing logic:

* BMW New (AUD): pulled from existing pricing catalogue
* Our Price (AUD): default to 50% of BMW new price unless existing used price data exists
* Saving: BMW New minus Our Price
* Category: catalogue category
* BMW OEM #: matched OEM part number

## Quote Review Screen

Before exporting, show a review table where the vendor can:

* Edit part description
* Edit OEM number
* Edit BMW new price
* Edit our price
* Remove incorrect parts
* Add missing parts manually
* Mark items as “needs review”

## Data Storage

Store each generated quote with:

* Quote ID
* Vendor ID
* Customer ID if available
* VIN
* Vehicle
* Uploaded photos
* AI damage analysis JSON
* Final quote rows
* CSV export URL
* Created date
* Last updated date

## Important Notes

Use the attached G80 M3 CSV and screenshot as the reference output format.

Use the attached damaged G80 M3 photos as the reference test case.

The first working version does not need to be perfect. The priority is:

1. Upload photos
2. Run AI damage analysis
3. Match likely damaged parts
4. Generate editable quote
5. Export CSV in the existing format

Build this as a workflow tool for paid vendors, not a public free tool.
