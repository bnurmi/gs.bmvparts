## Quick start (for Replit)

1. **Drop tokens in.** Copy `bmv-tokens.css` into `client/src/index.css` (or import it as the first stylesheet). Copy `tailwind.config.js` to your project root, merging with any existing config.

2. **Add the theme toggle.** Inline `theme-toggle.js` into your HTML `<head>` BEFORE any CSS — this prevents flash-of-unstyled-content on page load.

3. **Place logo files.** Put all `.svg` files into `client/public/logos/` (or wherever your static assets live). Update the sidebar/header to reference `/logos/bmv-logo-header.svg` and `/logos/bmv-logo-header-dark.svg`. The CSS variables `--logo-url` and `--logo-monogram-url` already point to those paths.

4. **Hand the markdown spec to the agent.** Push `BMV-BRAND-SPEC.md` to your repo root and tell Replit's agent: *"Implement the brand spec in BMV-BRAND-SPEC.md across the codebase, starting with tokens, then sidebar/logo, then homepage hero, then car cards, then part detail active states. Dark mode last."*

The spec is intentionally high‑level — it tells the agent what to do, not how to do it, so it can use context-aware judgement.
